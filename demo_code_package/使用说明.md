# 演示代码生成逻辑使用说明

## 概述

`generate_demo_code.py` 包含了从 `app.py` 中提取的生成演示分区的完整逻辑。这个文件可以直接在 `app.py` 中使用，或者用于生成演示代码。

## 主要函数

### 1. `generate_demo_partition_from_call_graph()`

基于真实调用关系生成演示分区（对应 `app.py` 步骤2.7的逻辑）

**参数：**
- `call_graph`: 调用图字典 `{caller: Set[callees]}`
- `min_methods`: 最小方法数量（默认20）

**返回：**
- 演示分区字典，包含：
  - `partition_id`: 分区ID
  - `name`: 分区名称
  - `methods`: 方法列表
  - `paths_map`: 路径映射（按叶子节点分组）
  - `paths`: 路径列表
  - `internal_edges_count`: 内部调用关系数量
  - `leaf_nodes`: 叶子节点列表

**使用示例：**
```python
from demo_code_package.generate_demo_code import generate_demo_partition_from_call_graph

call_graph = {
    "method1": {"method2", "method3"},
    "method2": {"method4"},
    # ...
}

demo_partition = generate_demo_partition_from_call_graph(call_graph)
if demo_partition:
    print(f"生成演示分区: {demo_partition['name']}")
    print(f"方法数: {len(demo_partition['methods'])}")
    print(f"路径数: {len(demo_partition['paths'])}")
```

### 2. `generate_demo_partition_with_methods()`

基于方法列表生成演示分区（对应 `app.py` 步骤6.5.4的逻辑）

**参数：**
- `available_methods`: 可用方法列表（四段内部格式）

**返回：**
- 演示分区字典，包含：
  - `partition_id`: 分区ID
  - `name`: 分区名称
  - `methods`: 方法列表
  - `paths_map`: 路径映射
  - `paths`: 路径列表
  - `fqns`: FQMN信息列表
  - `inputs`: 输入参数列表
  - `outputs`: 返回值列表

**使用示例：**
```python
from demo_code_package.generate_demo_code import generate_demo_partition_with_methods

available_methods = [
    "analysis.function_node_enhancer.FunctionNodeEnhancer.identify_leaf_nodes",
    "analysis.path_level_analyzer.PathLevelAnalyzer.generate_path_level_cfg",
    # ...
]

demo_partition = generate_demo_partition_with_methods(available_methods)
print(f"生成演示分区: {demo_partition['name']}")
print(f"方法数: {len(demo_partition['methods'])}")
print(f"路径数: {len(demo_partition['paths'])}")
```

## 辅助函数

### `find_connected_component()`

使用BFS找到包含start_node的连通分量

### `find_paths_from_leaf()`

从叶子节点回溯找到路径

## 在 app.py 中使用

可以在 `app.py` 中导入并使用这些函数：

```python
from demo_code_package.generate_demo_code import (
    generate_demo_partition_from_call_graph,
    generate_demo_partition_with_methods
)

# 在步骤2.7中使用
demo_partition = generate_demo_partition_from_call_graph(call_graph)
if demo_partition:
    partitions.insert(0, demo_partition)
    demo_partition_created = True

# 在步骤6.5.4中使用
if not has_valid_partition:
    demo_partition = generate_demo_partition_with_methods(available_methods)
    partitions.insert(0, demo_partition)
```

## 注意事项

1. 方法签名必须符合四段内部格式：`包.文件.类.方法`
2. 调用图必须包含真实的调用关系
3. 生成的路径会自动验证，确保所有方法都在分区内
4. 路径长度限制在2-4之间，符合功能路径约束
