# 代码结构分析工具 - 运行指南

## 功能说明

这个应用可以：
1. ✨ **选择本地项目文件夹** - 通过浏览器选择要分析的项目目录
2. 📊 **实时显示分析进度** - 任务进度条动态更新，告知用户分析预计完成时间
3. 🎨 **生成交互式可视化图表** - 使用Cytoscape.js显示代码结构关系图
4. 🔍 **支持交互操作** - 缩放、拖拽、搜索节点

## 安装依赖

```bash
pip install flask
```

## 运行应用

### 方式1：直接运行Flask应用（推荐）

```bash
python app.py
```

然后在浏览器中访问：
```
http://127.0.0.1:5000
```

### 方式2：使用已生成的可视化

如果只想查看示例结果（无需后端），可以直接打开：
```
output_analysis/visualization_complete.html
```

## 使用流程

1. **打开应用**：访问 `http://127.0.0.1:5000`

2. **选择文件夹**：
   - 点击"选择文件夹"按钮
   - 选择包含代码的本地项目目录

3. **等待分析**：
   - 应用会显示实时进度条
   - 显示当前分析阶段信息
   - 进度从0%到100%变化

4. **查看结果**：
   - 分析完成后自动显示代码结构可视化图表
   - 支持：
     - 🔍 缩放和平移图表
     - 📌 点击节点查看详情
     - 🔄 重置图表布局

5. **返回重新分析**：
   - 点击"返回"按钮选择其他项目

## 图表说明

### 节点类型

- **蓝色节点** - 类定义
- **绿色节点** - 方法
- **红色节点** - 函数

### 节点信息

点击任何节点可查看：
- 名称和类型
- 所属文件
- 行号
- 返回类型
- 方法计数

## 技术栈

- **后端**：Flask（Python）
- **前端**：HTML5 + JavaScript + Cytoscape.js
- **分析**：Python AST（抽象语法树）
- **库**：networkx、ast

## 支持的项目类型

- ✅ Python项目（.py文件）
- 📋 项目结构分析
- 🔗 调用关系分析
- 🏗️ 继承关系分析

## 性能指标

- **节点数**：78个（示例项目）
- **边数**：359条（关系）
- **分析时间**：取决于项目大小
  - 小项目（<100文件）：2-5秒
  - 中等项目（100-500文件）：10-30秒
  - 大项目（>500文件）：30秒以上

## 常见问题

### Q: 分析进度卡住了？
A: 这可能是因为项目过大或文件过多。请检查终端是否有错误信息。

### Q: 可以分析其他语言的代码吗？
A: 当前版本只支持Python。未来可以扩展支持其他语言。

### Q: 图表显示不完整？
A: 尝试点击图表上的"重置"或"适应屏幕"按钮以调整视图。
