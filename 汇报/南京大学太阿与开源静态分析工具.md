# 南京大学太阿（Tai-e）是否开源 + 其他优秀开源静态代码分析工具

---

## 一、南京大学太阿（Tai-e）是开源的吗？

**是的，太阿（Tai-e）是开源的。**

- **项目名**：Tai-e（太阿）
- **定位**：面向 Java 的、易学易用的**程序分析框架**，由南京大学 Pascal 实验室开发，用于《静态程序分析》课程教学与实验。
- **开源与许可**：
  - **主框架**：GitHub 仓库 **pascal-lab/Tai-e**，约 1.8k stars、194 forks，文档站：https://tai-e.pascal-lab.net
  - **作业/教学版**：**pascal-lab/Tai-e-assignments**，约 1.1k stars、254 forks，基于“教学版 Tai-e”设计 8 个编程作业（A1–A8）
  - **许可**：通常为 **LGPL-3.0 / GPL-3.0** 等开源协议（具体以仓库 LICENSE 为准），可用于教育与研究。
- **特点**：相比 Soot、WALA 等老牌 Java 分析框架，Tai-e 设计更简洁，IR 简洁、接口清晰易扩展、文档友好，适合学习和二次开发。官方还计划发布“研究版 Tai-e”，提供更多分析和更好性能。

**结论**：太阿是开源的程序分析框架，可直接在 GitHub 上获取和使用。

## 二、还有哪些好、非常厉害的开源静态代码分析工具？

按“程序分析/编译器向”和“安全/多语言 SAST”分两类，便于对比选型。

### （一）程序分析 / 编译器向（深度分析、IR、数据流等）

| 工具 | 语言/平台 | 特点 | 开源与规模（约） |
|------|-----------|------|------------------|
| **Tai-e（太阿）** | Java | 易学、IR 简洁、教学+研究，污点分析等 | 开源，pascal-lab/Tai-e，~1.8k stars |
| **Soot** | Java / Android | 经典框架，多种 IR（Jimple 等），控制流/数据流/指针/污点分析，文档和社区成熟 | 开源，soot-oss/soot，历史悠久 |
| **WALA** | Java / JavaScript 等 | IBM 起源，类型系统、类层次、过程间分析，能力全面，学习曲线较陡 | 开源，wala/WALA |
| **Infer** | Java / C/C++/ObjC | Facebook，分离逻辑、空指针/资源泄漏等缺陷，CI 友好 | 开源，facebook/infer，~15k stars |
| **CodeQL** | 多语言（C/C++/C#/Java/JS/Python 等） | GitHub 官方，用“查询语言”写分析，安全研究社区活跃，漏洞规则可共享 | 开源，github/codeql，~7k stars |

这类工具偏“底层程序分析”，适合做调用图、数据流、污点、指针分析等，和你在做的代码分析（调用图、CFG、DFG 等）在技术栈上更接近。

### （二）SAST / 多语言 / 规则与 CI 向（找 bug、规范、漏洞）

| 工具 | 语言/平台 | 特点 | 开源与规模（约） |
|------|-----------|------|------------------|
| **Semgrep** | 多语言 | 规则轻量、可定制、CI 集成好（GitHub Actions 等），平衡易用与能力 | 开源，semgrep/semgrep，~10k stars |
| **SonarQube** | 多语言 | 质量/安全/重复代码，有社区版免费，企业版收费 | 社区版开源，~8k stars |
| **PMD** | Java 等 | 缺陷、死代码、风格，内置+自定义规则 | 开源，pmd/pmd，~4.7k stars |
| **Bandit** | Python | Python 安全扫描，易集成 | 开源，PyCQA/bandit，~5.9k stars |
| **Brakeman** | Ruby on Rails | Rails 安全专项 | 开源，~6.9k stars |
| **PHPStan** | PHP | PHP 静态分析，社区活跃 | 开源，~12.5k stars |
| **OWASP Dependency-Check** | Java / .NET 等 | 依赖漏洞扫描（SCA） | 开源，~5.9k stars |

这类更偏“直接找问题、上 CI”，和“深度程序分析框架”是互补关系。

## 三、和你项目的关联（简要）

- **Tai-e**：若将来做 **Java** 仓库的“学”（调用图、数据流、污点等），可考虑用 Tai-e 做底层分析引擎或对比实验；当前你是 Python 为主，可作技术储备或对比参考。
- **Soot / WALA**：Java 生态里做深度静态分析的标准选项，能力最强但复杂度高；Tai-e 文档里也常与二者对比。
- **CodeQL / Semgrep**：若要在“学”的基础上加一层**安全/规范检测**（如漏洞模式、坏味道），可考虑集成或借鉴规则思路；你现有分析以结构和行为为主，这类工具可作扩展方向。

---

**一句话**：南京大学太阿（Tai-e）是开源的 Java 程序分析框架；其他非常厉害的开源静态分析还有 Soot、WALA、Infer、CodeQL（偏深度分析），以及 Semgrep、SonarQube、PMD、Bandit 等（偏 SAST/多语言/CI）。
